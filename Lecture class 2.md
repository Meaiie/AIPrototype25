# Command Line พื้นฐานบน Terminal
### 1. list คำสั่งที่ใช้สำหรับแสดงแฟ้มข้อมูล
แสดง file/folder ที่อยู่ใน folder ปัจจุบัน
```
ls
```
แสดง Permission
```
ls -l
```
เรียกดูรายชื่อไฟล์และโฟลเดอร์แบบละเอียด
```
ls -lh
```
เรียงไฟล์ตามเวลาที่แก้ไขล่าสุด โดยเอาไฟล์ใหม่ที่สุดไว้ล่างสุด
```
ls -ltr
```

### 2. Change directory
กลับไปที่ Home
```
cd
```
ถอยออกมาจากโฟลเดอร์ปัจจุบัน 1 ครั้ง
```
Cd ..
```
ถอยออกมาจากโฟลเดอร์ปัจจุบัน 2 ครั้ง
```
Cd ../..
```
ขอดูชื่อโฟล์เดอร์และไฟล์ที่อยู่ในโฟลเดอร์นั้น ๆ
```
Cd .. หรือ <ชื่อโฟลเดอร์> / → Tab
```
เข้าไปในโฟล์เดอร์ที่ต้องการ
```
cd ../../mnt/c/<ชื่อโฟลเดอร์>
```

### 3. file/Folder Management
สร้างโฟล์เดอร์
```
mkdir <folder_name>
```
สร้างไฟล์
```
vi <ชื่อไฟล์>
```
แก้ไขไฟล์
```
vi i <ชื่อไฟล์>
```
>> Est แล้ว  :wq กดออกแล้วบันทึก<br>
>> Est แล้ว :q! กดออกแล้วไม่บันทึก<br>

ลบไฟล์
```
rm <ชื่อไฟล์>
```
ลบทุกอย่างในโฟล์เดอร์
```
rm -r <ชื่อโฟล์เดอร์>
```
### 4. Move file
วิธีที่ 1 อยู่ตรงไหนของเครื่องก็ได้ แต่เราต้องระบุที่อยู่ไฟล์ให้ถูก
```
mv  ที่อยู่ไฟล์ที่จะย้าย/<ชื่อไฟล์>   โฟล์เดอร์ปลายทาง/.
```
วิธีที่ 2 เข้าไปในโฟล์เดอร์ที่อยากจะย้ายเข้าไป แล้วใช้คำสั่ง
```
mv  ที่อยู่ไฟล์ที่ต้องการย้าย/<ชื่อไฟล์>   .
```
> . หมายถึงตรงนี้

### 5. Move file
Copy file
```
Cp ที่อยู่ไฟล์ที่จะคัดลอก/<ชื่อไฟล์>  ที่อยู่ปลายทาง/.
```
Copy Folder
```
Cp -r ที่อยู่โฟลเดอร์ที่ต้องการคัดลอก/<ชื่อโฟลเดอร์>  + . ที่อยู่ปลายทาง/.
```
> Path คือ path เมื่อเทียบกับจุดที่เราอยู่ ถ้าจะเข้าเริ่มที่บอกว่าอยู่ตรงไหนด้วยจุด . 
ถ้าจะไปข้างนอกใช้เป็น ../<ชื่อโฟล์เดอร์ที่จะเข้าไป> เพื่อออกไปแล้วค่อยเข้าไปในโฟล์เดอร์ที่ต้องการ แล้ว . บอกว่าจะมาวางตรงนี้
> เหมือนเปิดประตูเข้าตึกที่มีห้องหลายห้อง เราจะไปที่ไหนต้องทำตาม step เข้าออก ต้องคิดภาพให้ออกว่าเราทำไปแล้ว 1 step นี้แล้วตอนทำไปแล้วเราจะอยู่ตรงไหน แล้วควรไปไหนต่อ
